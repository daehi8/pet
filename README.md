# 동물병원 리뷰 사이트

# 설명
- 영수증 인증을 통한 허위리뷰나 광고리뷰가 아닌 실사용자 리뷰만을 모은 사이트
- 각각의 동물병원마다 다른 가격대를 사용자입장에서 쉽게 비교하고 확인

# 개발목적
- 동물 진료가격 정보 공유
- 병원에 대한 사전정보 공유

# 개발환경
- WS : Tomcat 8.5
- DBMS : Oracle 11
- TOOL : Eclipse
- FRAMWORK : Spring 4, Mybatis, BootStrap
- API/LIB : Maven, javaxMail, KakaoAPI
